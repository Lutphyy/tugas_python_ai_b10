## 📌 Overview

Dokumen ini berisi kesimpulan dari beberapa tugas Python yang mencakup konsep dasar hingga penggunaan library populer seperti NumPy dan pandas.

---

## 🧩 Tugas 4 – Struktur Data & Manipulasi

Pada tugas ini, saya mempelajari dan mengimplementasikan berbagai struktur data di Python, seperti:

* List (akses, manipulasi, slicing)
* Tuple (immutability & unpacking)
* Set (operasi himpunan dan keunikan data)
* Dictionary (key-value dan iterasi)

Selain itu, juga digunakan:

* Nested structure (list of dictionary)
* Comprehension (list, dict, set)
* Pengecekan keanggotaan data

💡 **Kesimpulan:**
Struktur data sangat penting dalam pengolahan data karena memudahkan penyimpanan, pencarian, dan manipulasi data secara efisien.

---

## ⚙️ Tugas 5 – Function & OOP

Tugas ini berfokus pada:

* Pembuatan function dengan parameter dan return value
* Penggunaan default parameter
* Pembuatan class `Student` dengan atribut dan method
* Konsep OOP seperti encapsulation dan penggunaan `__str__`

💡 **Kesimpulan:**
Function membantu membuat kode lebih modular dan reusable, sedangkan OOP memudahkan pengelolaan data dan logika dalam bentuk objek.

---

## 📊 Tugas 6 – NumPy, pandas & File Handling

Pada tugas ini, saya menggunakan:

* NumPy untuk pengolahan data numerik (mean, median, std, dll)
* pandas untuk membuat dan mengelola DataFrame
* File I/O untuk menyimpan hasil ke file `.txt`
* Class `GradeBook` untuk mengelola data secara terstruktur

💡 **Kesimpulan:**
Library seperti NumPy dan pandas sangat powerful dalam pengolahan data, sementara file handling memungkinkan hasil analisis disimpan dan digunakan kembali.

Dari ketiga tugas ini, saya memahami bahwa:

* Python memiliki struktur data yang fleksibel dan mudah digunakan

* Library seperti NumPy dan pandas sangat membantu dalam pengolahan data skala lebih besar
* Penggabungan semua konsep ini menghasilkan program yang lebih efisien dan terstruktur
